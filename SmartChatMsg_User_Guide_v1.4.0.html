SmartChatMsg User Guide (v1.4.0)

UI Updates

  • Auto Populate Chat on Zone replaces the previous wording.
  • The Cooldown (mins) field is now aligned to the right of the checkbox.
  • Auto-Remove Pending Chat After (secs) has been moved to General settings.

Behavior

  • The revert timer is now global and applies to all commands.
  • Values under 30 seconds or empty values default to 60 seconds.
  • Settings automatically save when changed.
