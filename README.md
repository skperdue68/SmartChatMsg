# SmartChatMsg (v1.4.0)

ESO Add On For Handling Sending longer messages via a command

## What's New in 1.4.0
- Renamed "Auto Send Message to chat on Zone" to **Auto Populate Chat on Zone**
- Moved cooldown setting next to the checkbox for better layout
- Moved "Auto-Remove Pending Chat After (secs)" to General settings
- Setting is now global (account-wide)
- Automatically saves on change
- Enforces minimum of 30 seconds (defaults to 60 if invalid)

## Features
- Dynamic custom slash commands
- Per command + guild message storage
- Chat channel auto-restore
- Import/export support
