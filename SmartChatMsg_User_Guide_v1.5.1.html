SmartChatMsg User Guide (v1.5.1)

Behavior Settings

  • Repeat Every (mins) is stored per Command + Guild.
  • Retry Delay (mins) is stored per Command + Guild.
  • Retry Delay values of 0, blank, or invalid skip retry and resume the normal repeat cycle after the populate timeout.
  • If Retry Delay is higher than Repeat Every, it is automatically reduced to match Repeat Every.
  • Auto Populate Chat on Zone is stored per Command + Guild.
  • Cooldown (mins) is stored per Command + Guild.
  • Populate Sound is stored per Command + Guild and defaults to DUEL_START.
  • The Preview button plays the currently selected populate sound.

Populate Sound

  • The dropdown includes None plus a curated list of useful ESO sounds.
  • DUEL_START is always included.
  • When set to None, no sound is played.
  • The selected sound plays when text is automatically populated into the chat input by Auto Populate on Zone or Repeat.

General Behavior

  • The revert timer is global and applies to all commands.
  • If the timeout is reached, SmartChatMsg restores the prior chat channel and clears the pending chat buffer.
  • Parameters g1-g5 and o1-o5 are treated the same as 1-5.
  • Settings automatically save when changed.
